## Manifest

| Variable name     | Description                                   |
|-------------------|-----------------------------------------------|
| iso2_o | ISO-3166-1 2-letter code of exporter country |
| iso2_d | ISO-3166-1 2-letter code of importer country |
| year | Year in `YYYY` format |
| p | Pólya Index |
